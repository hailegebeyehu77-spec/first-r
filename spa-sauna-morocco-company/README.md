# Spa, Sauna, and Morocco Company

Welcome to the Spa, Sauna, and Morocco Company project! This application is designed to showcase our services and provide information about our offerings in a user-friendly manner.

## Project Structure

The project is organized as follows:

```
spa-sauna-morocco-company
├── public
│   ├── index.html          # Main HTML document for the application
│   └── manifest.json       # Metadata for the web application
├── src
│   ├── components          # Contains all React components
│   │   ├── Navbar.js       # Navigation bar component
│   │   ├── About.js        # About section component
│   │   ├── Menu.js         # Menu section component
│   │   ├── Services.js     # Services section component
│   │   └── Contact.js      # Contact section component
│   ├── App.js              # Main application component
│   ├── index.js            # Entry point of the React application
│   └── styles.css          # CSS styles for the application
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features

- **Responsive Navigation Bar**: Easily navigate through the sections of the application.
- **About Section**: Learn more about our company and its mission.
- **Menu Section**: View samples of our work and offerings.
- **Services Section**: Detailed descriptions of the services we provide.
- **Contact Section**: Get in touch with us through our contact information and Google Maps integration.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/hailegebeyehu77-spec/first-r.git
   ```

2. Navigate to the project directory:
   ```
   cd spa-sauna-morocco-company
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.
